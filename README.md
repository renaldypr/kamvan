# kanban
A kanban board web app, build with Vue.js and Firebase. Give it a try at https://kanban.renaldypratama.xyz/

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
